## Ambiente de Desenvolvimento
Para que todos tenham o ambiente de desenvolvimento o mais parecido possível e evitar problemas, certifique-se de ter as ferramentas acima com as seguintes versões:

| Ferramenta | Versão |
| --- | --- |
| Git | A mais recente |
| NodeJS | v18.17.1 |

Para instalar as devidas versões, siga o passo a passo do arquivo [`dev-environment.md`]().

## Guia de Instalação
> Com as ferramentas devidamente instaladas, execute os comandos abaixo

### **1. Clonar repositório**
```bash
git clone https://github.com/campusativo/design-system.git
```

### **2. Entrar na pasta do projeto**
```bash
cd design-system
```

### **3. Instalar as dependências**
```bash
npm install
```
